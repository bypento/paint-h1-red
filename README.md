# Paint H1 Red

[![npm version](https://shields.io)](https://www.npmjs.com/package/@bypento/paint-h1-red)

A tiny utility to turn all `<h1>` tags red.

🔗 **[View on npm](https://www.npmjs.com/package/@bypento/paint-h1-red)**

## Installation

```bash
npm install @bypento/paint-h1-red
```

## Usage

```javascript
import { makeH1Red } from '@bypento/paint-h1-red';

makeH1Red();
```
